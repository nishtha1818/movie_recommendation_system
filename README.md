# 🎬 AI Movie Recommendation System

## Overview
This project is an AI-based Movie Recommendation System that recommends five similar movies based on the movie selected by the user. It uses Content-Based Filtering and displays movie posters using the TMDB API.

## Features
- Recommends 5 similar movies
- Displays movie posters
- Simple and user-friendly interface
- Built using Machine Learning

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Requests
- TMDB API

## Dataset
- tmdb_5000_movies.csv
- tmdb_5000_credits.csv

## How to Run
1. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```
2. Run the project:
   ```
   streamlit run app.py
   ```
